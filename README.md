# OneFlesh – Christian Marriage App (UI Prototype)

This repository contains the UI design for **OneFlesh**, a Christian marriage app built to help couples grow emotionally, spiritually, physically, and mentally through weekly guided check-ins.

## 📱 Prototype Overview
The current prototype includes:
- **Dashboard Screen** – ESPM overview and weekly check-in entry point  
- **Weekly Check-In Screen** – Emotional, Spiritual, Physical, and Mental reflection questions  
- **Weekly Summary Screen** – Recap of responses with options to save or share
